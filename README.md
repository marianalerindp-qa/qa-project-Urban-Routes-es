# Proyecto Urban Routes



Comprobación de lista de pruebas para la página de Urban Routes para que el usuario pida un taxi.
Se especifica el modo comfort, junto con especificaciones de requerimientos y todos las campos de llenado correctos.


\##Tecnología

Se utilizó el siguiente ambiente:

* Pycharm
* GitHub
* Git Bash


\##Técnicas

* Se ocuparon distintos tipos de localizadores
* Importaciones de métodos vistos en los capitulos
* Esperas explicitas
* Modelo POM
